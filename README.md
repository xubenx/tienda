# Aluminios Dashboard — Análisis de Ventas

Dashboard Streamlit + SQLite para analizar ventas de una ferretería de aluminio (POS Eleventa).

## Estructura

- `dashboard.py` — Dashboard principal de ventas con gráficas interactivas
- `MI_BASE.sqlite` — Base de datos migrada desde Firebird
- `iniciar.sh` — Script de inicio para Linux
- `requirements.txt` — Dependencias Python

## Inicio rápido (Linux Mint)

```bash
chmod +x iniciar.sh
./iniciar.sh
```

Abre http://localhost:8501 en tu navegador.

## Requisitos

- Python 3.10+
- Las dependencias se instalan automáticamente con `iniciar.sh`
